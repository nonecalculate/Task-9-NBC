10번 과제 언리얼 플러그인 제작입니다. 따로 레포를 파기에는 애매해서 여기에 올리겠습니다. 코드와 패키징된 플러그인이 있습니다.

제 플러그인 주제는 게임 DarkAndDarker 베이스의 속성 기반 스텟 계산 플러그인입니다. 총 7 개의 속성이 있으며 이는 여러 스테이터스에 가산점을 줍니다.

Strength = 체력, 물리력

Vigor = 체력, 체력 재생력

Agility = 이동 속도, 공격속도

Dexterity = 공격 속도, 상호 작용 속도

Will = 주문력 , 버프 지속시간

Knowledge = 주문 시전 속도, 주문 회복 속도

Resourcefulness = 쿨타임

이는 struct 를 이용해 두개로 분리하여 하나는 속성, 하나는 스테이터스를 담당하게 코드를 설계했습니다. 사용하려면 캐릭터에 위와 같은 스텟과 속성이 필요합니다.
